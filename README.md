# 🩺 Diabetes Prediction Project

This is a simple beginner Data Science project to predict whether a person has diabetes or not.

---

## 📊 About the Project

* The goal is to build a basic machine learning model
* The model predicts diabetes using health-related data
* **Target variable:** `Outcome`

  * `0` → No Diabetes
  * `1` → Diabetes

---

## 🗂️ Dataset Features

Some common features used:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

---

## 🛠️ Tools Used

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## 🚀 Steps

1. Load the dataset
2. Clean the data
3. Split into training and testing sets
4. Train a model (e.g., Logistic Regression)
5. Make predictions

---

## 📈 Goal

To understand how machine learning can be used to predict diseases like diabetes.
